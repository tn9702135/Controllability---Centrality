This repository contains the MATLAB code and datasets for the paper titled "Exploring Relationships Between Controllability Measures and Centrality Metrics in Brain Structural Network" by Tahereh Niyazmand, Farid Sheikholeslam, and Marzieh Kamali.
The paper is published in Neuroscience Informatics and is available at: https://doi.org/10.1016/j.neuri.2026.100276
